# ruby-portfolio
Rubyの学習アウトプット
TrelloTaskの出力

### 目的
- ToDoリスト管理性の向上
- Rubyでのクラスメソッド、基本構文のアウトプット
### 達成によるメリット
- Todoリストによる個人レベルの進捗管理を取得することが出来る。（全体進捗が把握しづらいという個人的な意見）
### 実装内容
- TrelloAPIで認証キー、トークンの取得
- 任意で選択したボードからリストを指定し、設定されているカード情報を取得（今回は自身、Todo（やること）リストのカード情報を取得）
- カード情報からカードの'name'を取得、出力
### 改善内容
